# Spider of chips

## Introduction

Crawling [jr.jd](https://jr.jd.com) datas.

## Enviroment:

* Python version: `Python 2.7.13`
* Depend modules: `requests`, `BeautifulSoup`, `xlwt`

## Usage

Setting enviroment of above when you use it. `main.py` is main entry of project. And run command:

```
    python spider.py
```
